# Neovim Plugins
## Structure
```
.
├── init.lua
├── lua
│   ├── completions.lua
│   ├── configs
│   │   ├── dashboard.lua
│   │   ├── lspconfig.lua
│   │   ├── lualine.lua
│   │   ├── neotree.lua
│   │   ├── none-ls.lua

│   ├── keybinds.lua
│   ├── plugins.lua
│   └── settings.lua
└── .stylua.toml
```
There's quite alot of plugins, so make sure to check out `lua/plugins.lua` to view all of them.
